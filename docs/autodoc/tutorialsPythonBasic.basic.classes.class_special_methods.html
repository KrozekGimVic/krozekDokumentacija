tutorialsPythonBasic.basic.classes.class_special_methods module¶

An example using some of the Python special class methods.

class tutorialsPythonBasic.basic.classes.class_special_methods.Fraction(x, y)[source]¶

Bases: object

A simple fraction.

reduce()[source]¶

Reduce the fraction.

>>> frac = Fraction(2, 4)
>>> frac.reduce()
>>> print(frac)
1/2

>>> frac = Fraction(5, 8)
>>> frac.reduce()
>>> print(frac)
5/8

>>> frac = Fraction(-6, 9)
>>> frac.reduce()
>>> print(frac)
-2/3
value()[source]¶

Returns the value of the Fraction.

>>> Fraction(1, 2).value()
0.5
>>> Fraction(2, 4).value()
0.5
