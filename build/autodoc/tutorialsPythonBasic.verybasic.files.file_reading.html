tutorialsPythonBasic.verybasic.files.file_reading module¶

An example on how to read from a file. Keep in mind that the paths for opening the file are relative, so you need to run this file from inside the folder it is located in for the tests to work.

tutorialsPythonBasic.verybasic.files.file_reading.readFile(filename)[source]¶

Read the file named ‘filename’ and print the whole thing.
